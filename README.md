## Description

Collaborative platform for training courses

## Versions

**npm:** 7.12 (update `npm i -g npm@latest`)

**Angular:** 11.2

**Node:** 14.16

## Installation

```
git init
git remote add origin https://github.com/flix-flix/FormaSchool-Front
git pull origin main

npm i
ng serve -o
```
